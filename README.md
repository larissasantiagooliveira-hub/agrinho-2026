# agrinho-2026
Este projeto foi feito utilizando o Gemini IA, o prompt usado no projeto foi: elabore um projeto com as linguagens html, css e java script com o tema agro forte, futuro sustentável, equilibrio entre a produção e meio ambiente. Separe o html, css e java script. Na imagem de fundo do site coloque um trator, colheitadeira, com as cores verde e marrom, com as fontes Great Vibes e Classic.
No site temos o tema do agrinho desse ano "agro forte, futuro sustentável", os pilares do nosso site que são: produção consciente, equilíbrio vital e tecnologia verde. Simulador de impacto sustentável, que é colocado a quantidade de hectares que tem e aparece quanto você economizará de aguá.
